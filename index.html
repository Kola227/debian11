Установка Debian 11.5.0

Debian — операционная система, состоящая из свободного ПО с открытым исходным кодом. В настоящее время Debian GNU/Linux — один из самых популярных и важных дистрибутивов GNU/Linux, в первичной форме оказавший значительное влияние на развитие этого типа ОС в целом. Debian может использоваться в качестве операционной системы как для серверов, так и для рабочих станций

Ниже рассмотрим пошаговую установку debian 11.5.0.

Выбираем пункт "Install"

Выбираем пункт "English - English"

Выбираем пункт "United States"

Выбираем пункт "American English"

Задаем имя хоста (его можно будет изменить с помощь команды: echo "ИМЯ ХОСТА" > /etc/hostname, не забудьте перезагрузить машину.)

Пропускаем данный пункт, нажав кнопку Continue

Задаем пароль Root

В дистрибутивах Linux пользовательские привилегии предоставляются путем открытия доступа к root. Наличие таких прав позволяет использовать функциональность операционной системы более широко и выполнять необходимые действия в корневом каталоге.

Повторяем пароль

Задаем Имя аккаунта пользователя(Вводим пароль и повторяем его)

Выбираем временную зону "Eastern"

В разметке диска выбираем "Guided - use entire disk"

Разметка диска — это операция деления диска на секции. Каждая выделенная секция не зависит от других.

Выбираем диск для разметки

Выбираем пункт "All files in one partition"

Выбираем пункт "Finish partitioning and write changes to disk"

Выбираем пункт "Yes"

Выбираем пункт "No"

Выбираем пункт "United States"

Выбираем пункт "deb.debian.org" и жмем Continue

Выбираем пункт "No"

Пробелом убираем Debian desktop environment и GNOME, с помощью Tab переходим на Continue и жмем Enter

Выбираем пункт "Yes", устанавливаем GRUB

Выбираем пункт "/dev/sda"

Жмём Continue, мы успешно установили Debian 11.5.0

Большинство ошибок на этапе установки можно будет исправить уже после установки дистрибутива*
