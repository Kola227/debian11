Настройка репозитория

Репозиторий - это место в сети интернет, где хранятся какие-либо данные. Репозиторий операционной системы линукс - это место где хранятся программы (пакеты) этой операционной системы.

Для работы с репозиторием нам будет нужен выход в интернет.

В настройках виртуальной машины выбираем во вкладке сеть, тип подключения NAT или Сетевой мост. Запускаем машину.

Зайдем в машину под root, сменим имя хоста, выведем командой cat содержимое файла hostname.
root
вводим созданный нами пароль
echo dhcp > /etc/hostname (командой echo меняем содержимое файла hostname на "dhcp")
cat /etc/hostname (командой cat выводим содержимое файла hostname)
reboot (командой reboot перезагружаем машину, чтобы наши настройки применились)

Узнаем название нашего интерфейса, получим ip по dhcp, отредактируем "sources.list".
ip a (команда выводит информацию о cетевых интерфейсах)
В моём случае нужный мне сетевой интерфейс называется "enp0s3".

С помощью команды "nano /etc/network/interfaces" отредактируем файл "interfaces"
allow-hotplug enp0s3 заменим на:
auto enp0s3
iface enp0s3 inet dhcp
Cохраним (Ctrl+X, Y, Enter )

Перезагрузим службу networking, проверим получение ip по dhcp.
systemctl restart networking
ip a

С помощью команды "nano /etc/apt/sources.list" отредактируем файл "sources.list"
Закомментируем ненужные строки и напишем свою:
deb http://ftp.ru.debian.org/debian/ bullseye main
Cохраним (Ctrl+X, Y, Enter )

Командой "apt update" обновим пакеты нашей машины.
